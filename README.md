# rtcamp_Pune
Landing Page for RT Camp

  1. First section is header in which we have our logo and search bar.
  2. Second Section is about-section where carousel and small description is present.
  3. Third section has users sharing their experiences.
  4. Fourth section tells about glimpes of company's culture, news, weather, facebook followers, etc.
  5. Fifth section tells about the clients that the company serve its products to.
  6. And the last section is footer which has nice rich navigations to visit different pages of the company's website.

Technologies used to build the landing page :
1. HTML
2. SASS
3. Javascript
4. Bootstrap

SASS : Its a preprocessor of CSS which adds reusability feature to CSS and divides the styling code into diffrent 
logical code module files so that quick search is possible .Additional features like CSS variables and nested rules helps to write efficient code. And has lot more to explore.
