# rtcamp_Pune
Landing Page for RT Camp

First section : is header in which we have our logo and search bar.
Second Section : is about-section where carousel and small description is present.
Third section :  has users sharing their exeriences.
Fourth section : tells about glimpes of company culture, news, weather, facebook followers, etc.
Fifth section : tells about the clients that the company serve its products to.
And the last section  : is footer which has nice ricg navigations to visit diffrent pages of the company's website.

Technologies used to build the landing page :
1. HTML
2. SASS
3. Javascript
4. Bootstrap

SASS : Its a preprocessor of CSS which adds reusability feature to CSS and divides the styling code into diffrent 
logical code module files so that less searching and quick access is possible . And has lot more to explore.
